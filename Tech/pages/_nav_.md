title:: <nav>

- This tag is used to create the markup that defines the page navigation. Into this we typically add an `ul` or `ol` list:
- ```
  <nav>
      <ol>
          <li><a href="/">Home</a></li>
          <li><a href="/blog">Blog</a></li>
      </ol>
  </nav>
  ```
